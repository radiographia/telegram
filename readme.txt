==========================================
 РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ: Unicode Styler 1-a
==========================================

1. ЦЕЛЕВАЯ АУДИТОРИЯ
--------------------
Программа предназначена в первую очередь для ПРОГРАММИСТОВ, активно использующих мессенджеры и социальные сети.
Основная цель — расширить возможности стилизации текста при оформлении сообщений, постов и заметок.

Особенно удобно использовать Unicode-стили для:
 - выделения переменных, ключевых слов, фрагментов кода;
 - акцентирования чисел, терминов, латинских слов;
 - оформления заголовков и подписей.

Также программа будет полезна и обычным пользователям, желающим сделать текст выразительным и уникальным.


2. ПРЕИМУЩЕСТВА
---------------
 - Работает "из коробки", не требует установки и зависимостей;
 - Минимальная нагрузка на систему, быстрая работа;
 - Поддержка буфера обмена: можно сразу вставить результат в любое приложение;
 - Имеется окно предпросмотра стилизованного текста.


3. СОВМЕСТИМОСТЬ
----------------
Программа протестирована на следующих версиях Windows:
 - Windows 7
 - Windows 8 / 8.1
 - Windows 10
 - Windows 11


4. ШРИФТЫ ДЛЯ ПРЕДПРОСМОТРА
---------------------------
Для корректного отображения всех символов в предпросмотре рекомендуется установить шрифты:

 - DejaVuSans.ttf
 - DejaVuSans-Bold.ttf
 - DejaVuSans-BoldOblique.ttf
 - DejaVuSansMono.ttf
 - DejaVuSans-Oblique.ttf

Скачать можно отсюда:
 https://sourceforge.net/projects/dejavu/files/dejavu/2.37/dejavu-fonts-ttf-2.37.zip/download

>> Без этих шрифтов программа будет работать, но предпросмотр может быть искажен.


5. ОБЗОР ИНТЕРФЕЙСА И ФУНКЦИЙ
-----------------------------
 - Элемент "Тема" — переключает тему оформления (Светлая / Тёмная);
 - Поле ввода (верхнее) — введите исходный текст;
 - Поле вывода (нижнее) — отображает стилизованный результат (только для чтения);
 - Выбор "Charset" — определяет стиль кодировки: Fullwidth, Superscript, Math Bold и др.;
 - Символ отступа — выбор невидимого символа разной ширины;
 - Кнопка "Копировать" — копирует результат в буфер обмена;
 - Кнопка "Очистить" — очищает текстовые поля;
 - Кнопка "О программе" — показывает это руководство;
 - Ширина отступа — задаёт количество невидимых символов в начале строки (если строка начинается с пробела).


ВАЖНО!
------
Unicode-стилизация — это не смена шрифта, а ПРЕОБРАЗОВАНИЕ СИМВОЛОВ.
Такой текст не является исполняемым кодом. Он предназначен исключительно для
визуального представления в соцсетях, мессенджерах и т.д.
